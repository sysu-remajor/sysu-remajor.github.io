
## 向项目提交 pull request

> [!tip]
> 如果你是（或者未来是）一个计算机学生的话，请务必学会使用 github 进行项目协作。

首先注册并登陆 github 账户，打开 [sysu-remajor.github.io](https://github.com/sysu-remajor/sysu-remajor.github.io) 点击 `Fork` 按钮，在你的 repo 上进行修改后，向项目提交 pull request。

又或者，你愿意长期参与维护这个项目，发邮件到 sysu-remajor@proton.me，附上你的 github user name 和 email，申请加入维护。

## 使用其他方式提交贡献

向项目的维护者，比如 candlest （你可以在 qq 微信或者其他平台联系到他）发送信息，他们会协助你进行贡献。
