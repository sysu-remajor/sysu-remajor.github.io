# Frontmatter Template Snippets

Use these snippets at the top of markdown notes.

## Minimal Note

```yaml
---
title: Page Title
tags: [note]
draft: false
---
```

## Article Page

```yaml
---
title: Article Title
description: One-line summary for preview cards.
tags:
  - article
  - topic
draft: false
lang: zh-CN
enableToc: true
comments: true
created: 2026-02-24
modified: 2026-02-24
published: 2026-02-24
---
```

## Project Page

```yaml
---
title: Project Name
description: What this project does.
tags:
  - project
aliases:
  - old-project-name
permalink: /project/project-name
socialDescription: Share card description.
socialImage: /images/project-cover.png
cssclasses:
  - wide
draft: false
---
```
